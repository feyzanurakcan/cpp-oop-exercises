# C++ OOP Exercises

This repository contains my C++ practice examples focused on object-oriented programming concepts.

## Topics Covered
- Static member variables
- Object arguments
- Friend functions
- Class design and object interaction

## Included Examples
- **BankAccount Static Example**  
  Demonstrates how to use a static member variable to track the total number of created accounts.

- **BankAccount Transfer Example**  
  Simulates money transfer between two bank account objects.
  
- **FinancialApplication Static Example**  
  Shows how a shared interest rate can be managed using a static class member.

- **Funds Transfer with Friend Function**  
  Demonstrates money transfer between two different bank classes using a friend function.

## Purpose
I created this repository to organize my coursework exercises and strengthen my understanding of core object-oriented programming concepts in C++.
